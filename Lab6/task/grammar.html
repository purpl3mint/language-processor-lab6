Грамматика была приведена полностью в постановке задачи, ее описание приведено ниже:

G[E]:
E→TA
A→ε|+TA|-TA
T→OB
B→ε|*OB|/OB
O→num|id|(E)

num - числовая константа Ц{Ц}
id - идентификатор Б{Б|Ц}
Б - [a, b, c, ..., z, A, B, ..., Z], Ц - [0, 1, 2, ..., 9]
