Множество состояний ДКА для решения третьей лабораторной работы

Постановка задачи

Разработать детерминированный конечный автомат, проверяющий выражение вида: 012 | 01(12)*32*.
Обозначения:
| - или;
* - любое количество повторений элемента, включая его отсутствие (замыкание Клини);
() - незначащие символы, выделяют часть выражения, к которой относится операция *.

Множество состояний ДКА

q1 - начальное состояние
q2 - прочитали 0 как первый символ
q3 - прочитали 1 как второй символ
q4 - прочитали 1 как символ, стоящий на 3+2n позиции, где n = 0, 1, ...
q5 - прочитали 2 как символ, стоящий на 4+2n позиции, где n = 0, 1, ...
q6 - конечное состояние, если 3 прочитана либо как третий символ, либо как символ, стоящий на позиции 5+2n, где n = 0, 1, ...
q7 - конечное состояние, если прочитала 2 как третий символ

Входной алфавит ДКА

Входной алфавит А = {0, 1, 2, 3}
