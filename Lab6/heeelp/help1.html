Описание функций меню

    Файл:
  1. Создать (Create) - создание нового документа (Ctrl+N)
  2. Открыть (Open) - открытие документа (Ctrl+O)
  3. Сохранить (Save) - сохранение документа (Ctrl+S)
  4. Сохранить как (Save as) - сохранение нового документа (Ctrl+Shift+S)
  5. Выход (Exit) - закрытие редактора (Alt+F4)
    Правка:
  1. Отменить (Undo) - отмена последнего действия (Ctrl+Z)
  2. Повторить (Redo) - повторение последнего действия (Ctrl+Y)
  3. Вырезать (Cut) - помещение выделенного фрагмента в буфер обмена и удаление его из документа (Ctrl+X)
  4. Копировать (Copy) - помещение выделенного фрагмента в буфер обмена (Ctrl+C)
  5. Вставить (Paste) - вставка фрагмента из буфера обмена (Alt+V)
  6. Удалить (Delete) - удаление фрагмента
  7. Выделить всё (Select all) - выделение всего содержимого (Ctrl+A)
    Текст:
  1. Постановка задачи
  2. Грамматика
  3. Классификация грамматики
  4. Метод анализа
  5. Диагностика и нейтрализация ошибок
  6. Тестовый пример
  7. Список литературы
  8. Исходный код программы
